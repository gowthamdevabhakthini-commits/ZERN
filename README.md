#  ZERN

### Next-Generation Smart Glasses for Brain-Computer Interaction

> A wearable Brain-Computer Interface (BCI) research project that combines embedded systems, biomedical signal acquisition, and intelligent software to create smart glasses capable of interpreting human bio-signals for future human-computer interaction.

---

##  Overview

ZERN is an ongoing engineering and research project focused on developing a wearable Brain-Computer Interface (BCI) integrated into smart glasses.

The project explores how Electroencephalography (EEG) and Electrooculography (EOG) signals can be captured, processed, and utilized to enable intuitive interaction between humans and computers.

The repository contains firmware, CAD designs, signal processing utilities, hardware resources, datasets, and experimental results collected throughout the development process.

---

#  Objectives

* Design a wearable BCI platform
* Acquire high-quality EEG and EOG signals
* Process bio-signals in real time
* Develop embedded firmware for wireless data acquisition
* Design custom smart-glasses hardware
* Build a scalable platform for future AI-based signal interpretation
